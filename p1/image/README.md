# Изображения: классификация ImageNet (PyTorch / torchvision)

## ТЗ (кратко)

**Задача:** для каждого изображения вывести топ-k классов ImageNet.  
**Вход:** все файлы `.jpg`, `.jpeg`, `.png`, `.webp`, `.bmp` в `input/` (по алфавиту). Если файлов нет — один прогон с синтетическим градиентом.  
**Выход:** `output/result.json` — массив `results`: у каждого файла `source` и `top_k`.  
**Ограничения:** CPU; веса подтягиваются при первом запуске (кэш `torch_cache` в compose). ImageNet обучался на **фотографиях**; для **живописи** и старых репродукций топ-классы часто остаются размытыми — это ограничение датасета, не только модели.

## Модель

`torchvision.models.vit_b_16` с весами `ViT_B_16_Weights.IMAGENET1K_V1`. Инференс: **TenCrop** (10 кропов 224×224 после уменьшения короткой стороны до 256), усреднение логитов — стабильнее одного center crop.

## Сборка и запуск

```bash
cd p1
docker compose build image
docker compose run --rm image
```

Положите один или несколько снимков в `image/input/`.
